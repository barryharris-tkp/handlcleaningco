# Phase 03: ChromaDB Vector Store + Semantic Retrieval

This phase adds the retrieval layer to Sonny — the ability to search past conversations by meaning, not just keywords. ChromaDB stores vector embeddings of session summaries generated by Ollama, and a new search API lets the frontend query them. By the end of this phase, Sonny has a working semantic search: you can type a question and find relevant past conversations, even if they used completely different words.
